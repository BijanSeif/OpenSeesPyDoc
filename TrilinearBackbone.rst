.. function:: hystereticBackbone('Trilinear', backboneTag, e1, s1, e2, s2, e3, s3)

   ``backboneTag`` |int|                 integer tag identifying the backbone function.

   ``s1`` |float|                        parameter :math:`s_1`.
   ``e2`` |float|                        parameter :math:`e_2`.
   ``s2`` |float|                        parameter :math:`s_2`.
   ``e3`` |float|                        parameter :math:`e_3`.
   ``s3`` |float|                        parameter :math:`s_3`.
